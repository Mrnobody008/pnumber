# pnumber
a simple script to convert numbers into Persian numbers

-------------------------------------------------------------------
USAGE


1 - add "pnumber.js" to your index html file
2- use tag <code><pnumber num="YOUR NUMBER"></pnumber></code>

exp : 

   <h1>
      my number is <code><pnumber num="2022"></pnumber></code>
   </h1>
