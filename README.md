# pnumber
a simple script to convert numbers into Persian numbers

-------------------------------------------------------------------
USAGE


1 - add "pnumber.js" to your index html file
2- use tag <pnumber num="YOUR NUMBER"></pnumber> 

exp : 

   <h1>
      my number is <pnumber num="2022"></pnumber> 
   </h1>
