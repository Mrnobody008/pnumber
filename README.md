# pnumber
a simple script to convert numbers into Persian numbers

-------------------------------------------------------------------
USAGE


1 - add "pnumber.js" to your index html file
2- use tag &lt;pnumber num="YOUR NUMBER"&60;&lt;/pnumber&60;

exp : 

   &lt;h1&60;
      my number is &lt;pnumber num="2022"&60;&lt;/pnumber&60;
  &lt;/h1&60;
